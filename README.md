# 🚀 Go DevOps Pipeline Project

A production-ready Go application demonstrating a complete DevOps workflow including containerization, CI/CD, and Kubernetes deployment.

## 🔧 Tech Stack

- Golang
- Docker
- Jenkins (CI/CD)
- Kubernetes
- HTML (UI)

## 📦 Features

- Lightweight Go web service
- Dockerized application
- Jenkins pipeline for automated build & deploy
- Kubernetes manifests for orchestration
- Scalable and production-ready setup

## 🏗️ Architecture

User → Go App → Docker → Jenkins CI/CD → Kubernetes Cluster
